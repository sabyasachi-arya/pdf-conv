🚀 Deploy this app on your server
1
Install LibreOffice: sudo apt install libreoffice (Ubuntu/Debian) or brew install --cask libreoffice (Mac)
2
Install Python deps: pip install flask flask-cors gunicorn
3
Run locally: python app.py or in production: gunicorn -w 4 app:app
4
Deploy to any Linux VPS (DigitalOcean, Render, Railway, Fly.io) — the Dockerfile included handles everything automatically.
